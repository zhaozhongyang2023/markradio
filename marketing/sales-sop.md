## 六、常见售后问题速查

| 问题 | 答案 |
|------|------|
| 网页打不开 | "桌面模式打开终端，输入 `systemctl --user restart moodwave.service`，刷新浏览器" |
| 没有声音 | "检查音量，然后 `systemctl --user restart moodwave.service`" |
| 生成歌单失败 | "检查 Key：`cat ~/.config/moodwave/config.env`，确认 `AI_API_KEY=` 有值" |
| Decky插件找不到 | "Decky 里点⚙ → Reload Plugins" |
| 升级后出问题 | "`cd ~/moodwave && git pull && npm install --omit=dev && npm run build && systemctl --user restart moodwave.service`" |
