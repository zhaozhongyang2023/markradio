## 五、常见售后问题速查

| 问题 | 答案模板 |
|------|---------|
| 网页打不开 | "桌面模式打开终端，输入 `systemctl --user restart moodwave.service`，然后刷新浏览器" |
| 没有声音 | "检查Steam Deck音量是否打开，然后 `systemctl --user restart moodwave.service`" |
| 生成歌单失败 | "检查AI Key是否正确：`cat ~/.config/moodwave/config.env`，确认 `AI_API_KEY=` 后面有值" |
| Decky插件找不到MoodWave | "确认插件目录存在：`ls ~/homebrew/plugins/moodwave-deck-companion/`。如果存在，Decky里点⚙ → Reload Plugins" |
| 想用网易云音乐 | "需要自建网易云API服务或者用第三方API地址。群里有教程（视情况补充）" |
| 升级后出问题 | "`cd ~/moodwave && git pull && npm install --omit=dev && npm run build && systemctl --user restart moodwave.service`" |

## 六、退款处理原则

- **安装前可退**：24小时内未安装可全额退款
- **安装后原则上不退**：描述中已写明「确认后不退」，但可根据情况协商部分退款
- **技术问题导致的退款**：优先排查解决，实在无法使用可全额退
- **闲鱼退款纠纷**：保留安装成功截图作为凭证

## 七、风控提示

- 不要用个人大号在闲鱼长期卖同款虚拟商品（易被AI检测下架）
- 建议2-3个闲鱼号轮换上架
- 商品描述避免出现「软件」「API」「激活码」「盗版」等敏感词
- 如被下架，换标题换号重新发布
- 定期清理聊天记录中涉及价格的敏感表述
