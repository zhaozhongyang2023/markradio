# DeepSeek API Key 申请教程（1分钟）

## 视频脚本 / 图文教程

---

### 三步搞定

#### 第一步：打开官网

浏览器打开：**platform.deepseek.com**

#### 第二步：注册账号

点右上角「注册」→ 用手机号或邮箱注册 → 验证码登录

> 完全免费，不需要绑定信用卡，不需要充值。

#### 第三步：创建 API Key

登录后 → 左侧菜单「API Keys」→ 点「创建新的 API Key」

随便起个名字（比如 `moodwave`）→ 点创建 → **立刻复制 Key**！

> ⚠️ Key 只显示一次，关掉页面就看不到了。
> 建议先粘贴到备忘录里备用。

---

### 费用说明

- DeepSeek 新用户赠送 **500万 tokens** 免费额度
- 生成一次歌单消耗约 5000-10000 tokens
- 免费额度够用 **500-1000 次**
- 额度用完可充值，最低 10 元起
- 不充值也完全不影响 MoodWave 正常使用（额度用完前都免费）
